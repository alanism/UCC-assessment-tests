# UCC x Renaissance STAR 360 - Reading Informational Craft & Structure

## Product Summary

Adaptive informational-reading instrument focused on author structure, signal words, purpose, and blueprint recognition in non-fiction passages.

## Intended Operator

Reading coaches and assessment operators who want a targeted scan of structural reading rather than broad comprehension review.

## Core Workflow

1. Start directly in testing mode.
2. Present informational passages and multiple-choice craft/structure questions.
3. Time each item at 60 seconds while adaptive difficulty moves with performance.
4. Finish and review the metrics tab.

## Inputs Collected

- Selected answer index
- Passage/question telemetry
- Response time and correctness
- Difficulty history across the session

## Outputs And Analytics

- Review-state metrics tab
- Item log and level history
- Session summary through `BrainAnalytics`

## AI Usage

- No live provider call is wired into this instrument in v1.
- The Local AI Settings panel stores operator-selected provider/model locally for this app origin only.

## Local Operation

- Install with `npm install`
- Run with `npm run dev`
