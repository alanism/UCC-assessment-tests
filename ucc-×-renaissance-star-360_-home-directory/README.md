# UCC x Renaissance STAR 360 - Home Directory

## Product Summary

Suite landing page that explains the STAR 360/UCC framing, presents informational operator panels, and links users into the deployed reading and math instruments.

## Intended Operator

Internal admins, evaluators, and coaches who need a single launch surface for the full suite plus contextual explanation of what each instrument is meant to measure.

## Core Workflow

1. Land on the directory overview.
2. Read the informational panels describing psychometric intent, pressure-based assessment, and recommended usage cadence.
3. Launch a reading or math instrument via the linked cards.

## Inputs Collected

- No learner identity or answer input is collected in current code.
- Current app state is informational and navigational only.

## Outputs And Analytics

- No telemetry export is generated by this app.
- Primary output is outbound navigation to deployed instruments.

## AI Usage

- No live provider calls are made in this app in v1.
- The Local AI Settings panel exists so operators can still save a preferred provider/model on this origin if needed for future suite expansion.
